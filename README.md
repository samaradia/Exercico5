# Polimorfismo com Interface em Java

Projeto simples em Java para demonstrar o conceito de **polimorfismo** utilizando **interfaces**.

## Conceitos abordados
- Interface
- Polimorfismo
- Classes e objetos
- `ArrayList`
- Laço `for-each`

## Descrição
O projeto define a interface `Forma`, que obriga as classes a implementarem o método `calcularArea()`.  
As classes `Circulo` e `Quadrado` implementam essa interface, cada uma calculando a área de sua própria forma.

No método `main`, os objetos são armazenados em uma lista do tipo `Forma`, e o Java chama automaticamente o método correto para cada objeto, demonstrando o polimorfismo na prática.

Projeto voltado para estudo e aprendizado de Java para iniciantes.
